# Currently project is abadond

## Dependecies
- npm
- composer
- docker-compose

## Installation

```
git clone https://github.com/Iviv122/LinkManager
cd LinkManager
npm install
composer install
cp .env.example .env
docker-compose up -d
```

## First Run
```
php artisan migrate
php artisan serve --host=0.0.0.0 --port=8000
```

## Run 
```
php artisan serve --host=0.0.0.0 --port=8000
```
